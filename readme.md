# keepMoving

Is a python script that draw some figures <del>on a previuosly open and maximized paint app window</del>. This prevents the monitor for going to sleep.

## Prerequisites

Python 3

- MODULES
  - pyautogui
  - time

## Usage

- Open paint and maximize it
- Run the Script

```bash
python keepMoving.py
```
